# Rcode_WhiskeredTern_activitybudgets
R scripts used for movement ecology analysis in my research on Whiskered terns conducted in ECOBIO (University of Rennes, France). 
This repository includes data processing, visualization, and statistical analysis code.
The scripts focus on behavioural classification, movement metrics computation, and statistical analyses.
